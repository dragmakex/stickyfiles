# (✯◡✯) stickyfiles

personal sticky OSX dotfiles as submodules:
- borders: window border config
- fastfetch: configs with luffy ascii art
- nvim: my nvim config based on NvChad
- sketchybar: OSX top bar setup and plugins
- skhd: hotkey configuration
- yabai: tiling window manager config

heavily inspired by Felix Kratz’s dotfiles: https://github.com/FelixKratz/dotfiles
